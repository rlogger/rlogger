# Rajdeep Singh

Graduate researcher at USC's Robotic Embedded Systems Laboratory (RESL), building multi-agent reinforcement learning systems, probabilistic hyperdimensional computing, physics-informed machine learning, and BCI tools in JAX.

Current work:

- [Bayes-HDC](https://github.com/rlogger/bayes-hdc): calibrated, differentiable hyperdimensional computing with conformal prediction and anomaly-detection guarantees.
- Opponent-aware model-based co-training for multi-agent reinforcement learning.
- Error-related-potential BCI pipelines and applied autonomy systems.
