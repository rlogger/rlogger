### Rajdeep Singh

Graduate researcher at USC's Robotic Embedded Systems Lab (RESL). I work on multi-agent reinforcement learning and physics-informed machine learning: systems that respect structure (other agents' strategies, physical law, calibrated uncertainty).

- **bayes-hdc**: probabilistic hyperdimensional computing in JAX. Calibrated probabilities and conformal prediction with finite-sample coverage guarantees. On PyPI (`pip install bayes-hdc`), archived at [doi:10.5281/zenodo.20635099](https://doi.org/10.5281/zenodo.20635099).
- **Now**: opponent-aware, model-based co-training for multi-agent RL.
- **Before**: AI safety for autonomous UAV swarms (NSF REU); interpretability and accountability at the AIEA Lab, UC Santa Cruz.
